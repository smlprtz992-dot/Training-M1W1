# Gym Age Classification Program

## Description

This project is a simple Python program that classifies people into gym groups according to their age.

This program is a basic example of conditional logic in Python.  
The script is a small decision system that determines which gym class a person should attend.

The program asks the user to enter their age and then evaluates the correct group.

## How it works

1. The program asks the user to enter their age using the input function.
2. The program converts the value into an integer.
3. The system checks different conditions using if and elif statements.
4. If the user is under 13, the program shows that they cannot enter the gym.
5. If the user is between 13 and 17, the program assigns the youth class.
6. If the user is between 18 and 59, the program assigns the general class.
7. If the user is 60 or older, the program assigns the senior class.

The system processes the conditions and displays the correct message.

Currently, the program **is running** a simple decision process to classify users by age.
